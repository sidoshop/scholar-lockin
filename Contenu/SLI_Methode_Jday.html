✅ Méthode J-Day
Ce que font les élèves qui réussissent — du premier coup d'œil à la dernière ligne
Ce guide répond à une seule question : par où je commence ? qu'est-ce que je lis en premier ? comment je sais quel doc va avec quelle question ? et si je ne comprends pas un mot ?
Toutes les stratégies sont tirées de l'analyse de 7 annales réelles (BAC 2021–2025).
1
Le protocole d'entrée — dans quel ordre lire le sujet
Les 5 étapes à faire avant d'écrire la moindre réponse · Protocole interactif
▶
Progression
0 / 17
1
Étape 1
Lis la présentation de l'entreprise
La première ou deuxième page — toujours
Note le nom de l'entreprise et son secteur d'activité
Tu sais dans quel univers tu vas travailler — gardes-les en tête pour tout le sujet
Identifie la problématique centrale (souvent en gras ou italique)
C'est la boussole de tout le sujet — toutes les questions y répondent indirectement
Repère les 3 dossiers et leur intitulé
Tu vois déjà les 3 grands thèmes — souvent : Stratégie / Gestion / Numérique&RSE
Ce que tu dois noter sur ton brouillon dès maintenant
Nom + secteur · Problématique centrale · Dossier 1 = ___ · Dossier 2 = ___ · Dossier 3 = ___
Étape : 0 / 3
2
Étape 2
Lis TOUTES les questions — PAS les documents
Les questions te disent ce que tu cherches dans les docs
Lis toutes les questions de A à Z sans t'arrêter pour chercher la réponse
L'objectif est juste de comprendre ce qu'on attend — pas encore d'y répondre
Encadre le verbe directeur de chaque question
Identifier / Analyser / Justifier / Calculer — chaque verbe a sa logique de réponse (voir Section 2)
Surligne les mots-clés notions dans chaque question
FRNG, flux tendus, RSE, compétences distinctives… ces mots te disent quelle notion du cours mobiliser
Marque les questions complexes ou floues d'un « ? »
Tu y reviendras — mais tu ne les oublies plus. Passer une question difficile n'est pas abandonner, c'est stratégique.
Pourquoi les questions AVANT les documents
Sans connaître les questions, tu lis les documents sans savoir quoi retenir. Avec les questions en tête, chaque phrase d'un document devient soit utile, soit inutile. Tu lis une seule fois, de façon active.
Étape : 0 / 4
3
Étape 3
Lis les titres des documents — PAS leur contenu
Les titres sont un code — apprends à les déchiffrer
Parcours les titres de tous les documents listés dans chaque dossier
Juste les titres — quelques secondes par document suffisent
Souligne les mots du titre qui résonnent avec les questions déjà lues
« éléments chiffrés » → question financière · « RSE / engagements » → Dossier 3 (voir Section 4)
Anticipe mentalement quel document ira avec quelle question
Tu n'as pas lu le contenu, mais tu devines déjà dans 70-80% des cas — tu affineras à l'étape 4
Exemple concret — BAC 2025 Bébé Serein
Q 2.1 : « Analyser l'équilibre financier (FRNG, BFR, TN) » → tu cherches un doc avec des chiffres.
Document 9 : « Éléments chiffrés de Bébé Serein au 31/12/2023 » → BINGO. Tu sais déjà, sans l'avoir lu.
Étape : 0 / 3
4
Étape 4
Fais le lien Question ↔ Documents sur ton brouillon
L'étape clé — elle te fait gagner sur toute la suite
Pour chaque question, note sur le brouillon les numéros de documents probables
Q 1.1 → docs 1, 2, 6 · Q 2.1 → doc 9 · Q 3.1 → docs 10, 11, 12
Vérifie quels documents sont LISTÉS dans l'énoncé pour chaque dossier — et utilise-les TOUS
Ignorer un document listé = perdre des points garantis. Si tu ne vois pas le lien, cite au moins 1 phrase.
Repère le document avec des colonnes de chiffres → c'est TOUJOURS la question financière
FRNG / BFR / TN ou coût de revient — mets ce doc de côté pour cette question précise
Ce que doit ressembler ton brouillon après cette étape
Q 1.1 → docs 1, 2, 6  |  Q 1.2 → docs 1, 3, 4  |  Q 2.1 → doc 9 (financier)  |  Q 3.1 → docs 10, 11, 12, 13 (RSE)
Tu sais déjà TOUT — avant d'avoir lu un seul document en entier.
Étape : 0 / 3
5
Étape 5
Réponds question par question
Lire les docs AU FIL des questions — pas tout d'un coup
Commence par Q 1.1 et avance dans l'ordre — les questions se construisent
Les examinateurs conçoivent souvent les questions pour que les réponses s'enchaînent logiquement
Lis SEULEMENT les docs liés à la question que tu traites — au moment de la traiter
On ne lit PAS les 12 docs d'une traite. Avant Q 1.1 → lis docs 1 et 2. Avant Q 1.2 → lis docs 3 et 4.
Si tu bloques sur une question, passe à la suivante — reviens après
Une réponse partielle rapporte des points. Rester bloqué 20 min sur 3 points = une erreur stratégique.
Ne laisse aucune question sans réponse — même approximative
Un correcteur valorise une réponse partielle bien construite plutôt qu'une feuille blanche
Règle d'or du SDGN
Cas + Cours = points. Le cas sans le cours, c'est descriptif. Le cours sans le cas, c'est hors-sujet. Les deux ensemble, c'est l'excellence.
Étape : 0 / 4
Résumé du protocole : Présentation entreprise → Toutes les questions → Titres des docs → Lien Q↔docs sur brouillon → Répondre Q par Q en lisant les docs au fur et à mesure
⭐
Protocole validé
Tu as appliqué la méthode des élèves qui réussissent.
Maintenant, rends ta copie avec confiance.
2
Décoder les verbes directeurs — ce que chacun attend
Chaque question commence par un verbe. Ce verbe est un code — clique pour le déchiffrer.
▶
🔍
Identifier / Repérer
▼
Ce qu'on attend :
Lister des éléments tirés des documents avec leur référence (Doc X). Pas de développement. Pas de "parce que".
Exemple de réponse
"Les ressources intangibles identifiées sont : la marque, le savoir-faire et les brevets (Doc. 2)."
⚠️ Piège : développer. Si le verbe dit "identifier", s'arrêter après avoir nommé — chaque mot de plus est du temps perdu.
✗ Ne pas faire : "L'entreprise identifie ses ressources car elle souhaite…" → c'est une explication, pas une identification.
🏷️
Caractériser / Présenter
▼
Ce qu'on attend :
Qualifier avec un type ou une catégorie + citer la source (doc n°…) + application à l'entreprise. Structure : définition + éléments + exemple de l'entreprise du sujet.
Exemple de réponse
"D'après le document 2, il s'agit d'un modèle économique de type freemium : le service de base est gratuit, et les fonctionnalités avancées sont payantes."
⚠️ Piège : Lister sans structure. "Présenter" implique une organisation lisible — numérotée, ou avec des titres.
⚖️
Justifier
▼
Ce qu'on attend :
Donner des raisons précises (chiffres, faits, notions) qui prouvent un choix ou une affirmation. Formule : Oui/Non + parce que + selon le document…
Exemple de réponse
"Oui, l'entreprise adopte une démarche RSE, car elle a mis en place un programme de réduction de son empreinte carbone (doc. 3) et a amélioré la qualité de vie au travail de ses salariés."
⚠️ Piège : Reformuler la question autrement sans donner de vraies raisons.
💬
Expliquer
▼
Ce qu'on attend :
Développer le mécanisme de cause à effet. Formule : X entraîne Y, ce qui produit Z. Montrer un lien logique, pas juste décrire.
Exemple de réponse
"Le recours aux flux tendus (doc. 2) permet à l'entreprise de produire uniquement à la commande, ce qui réduit les stocks et diminue les coûts de stockage, améliorant ainsi la rentabilité."
⚠️ Piège : Rester descriptif. "Expliquer" = montrer un lien de cause à effet, pas juste constater.
🔬
Analyser
▼
Ce qu'on attend :
Si calcul : formule visible + calcul détaillé + résultat + commentaire OBLIGATOIRE. Si rédactionnel : décomposer + interpréter + commenter. Jamais de calcul sans commentaire.
Exemple de réponse (calcul)
"FRNG = Ressources stables − Emplois stables = 850 000 − 620 000 = 230 000 €
Interprétation : le FRNG est positif, ce qui signifie que les investissements à long terme sont intégralement financés par des ressources durables."
⚠️ Piège : Donner le résultat sans l'interpréter. La moitié des points est sur le commentaire.
🧮
Calculer
▼
Ce qu'on attend :
Écrire la formule → Poser les chiffres du cas → Résultat avec unité → Interprétation en 1 phrase. La formule visible = des points même si le calcul est faux.
Exemple de réponse
"Taux d'évolution = (Valeur finale − Valeur initiale) / Valeur initiale × 100
= (1 200 000 − 1 000 000) / 1 000 000 × 100 = +20 %
Le chiffre d'affaires a progressé de 20 %, traduisant une croissance significative de l'activité."
⚠️ Piège : Donner le résultat seul sans la formule ni l'interprétation = pertes de points assurées.
📝
Montrer / Démontrer
▼
Ce qu'on attend :
Construire un raisonnement avec des preuves tirées des documents ET du cours. Pas d'affirmation sans preuve ("car les docs le montrent" ne suffit pas).
Exemple de réponse
"La démarche qualité de Bébé Serein (AOP, ESAT, doc. 4 et 6) constitue un avantage concurrentiel durable, car elle permet de se différencier dans un marché fortement concurrentiel tout en valorisant son image de marque."
⚠️ Piège : Affirmer sans preuve ou sans citer le dossier. Chaque argument doit être suivi d'une référence documentaire.
💡
Proposer / Formuler
▼
Ce qu'on attend :
Inventer des éléments cohérents avec la situation de l'entreprise (indicateurs, solutions). On ne recopie PAS les documents — on imagine à partir du contexte.
Exemple de réponse
"Pour réduire son BFR, l'entreprise pourrait négocier des délais de paiement plus courts avec ses clients. Cette mesure diminuerait l'actif circulant et donc le besoin de financement lié à l'exploitation."
⚠️ Piège : Proposer sans justifier. "Je suggère X" sans "parce que Y" = réponse incomplète.
Cas particulier — "En 15 lignes"
Introduction (2 lignes) : définir la notion + annoncer le plan
Axe 1 (3-4 lignes) : idée + exemple issu de l'entreprise du sujet
Axe 2 (3-4 lignes) : idée + exemple d'une autre entreprise connue (Airbnb, Zara, Danone…)
Axe 3 (2-3 lignes) : nuance ou limite
Conclusion (1-2 lignes) : synthèse courte qui répond à la question globale

Les 5 règles d'or pour trouver le bon document
L'examinateur te donne les indices — apprends à les lire
▶
R1
L'énoncé te donne DÉJÀ la liste des documents à utiliser
Chaque dossier commence par : "Documents à exploiter : Document 1, 4, 7…"
▶
Chaque dossier liste explicitement les documents à utiliser. Ces documents ET SEULEMENT CES DOCUMENTS doivent être cités.
⚠️ Règle absolue : si l'énoncé t'indique d'utiliser les documents 3, 7 et 9 pour Q 2.1, tu DOIS utiliser ces 3 documents. Ignorer l'un d'eux = perdre des points garantis.
Si tu bloques sur un document : cherche au moins 1 élément à citer ("Selon le document X, …"). Les correcteurs vérifient que chaque document listé a été utilisé.

Dans toutes les annales analysées — sans exception
▶

Document factuel (chiffres, stats) → appuyer un argument quantitativement
Document témoignage ou presse → illustrer concrètement
Document institutionnel (site entreprise) → présenter la position officielle
Formule magique pour citer plusieurs documents
"Selon le Document 2, […]. Cette information est confirmée par le Document 5 qui précise que […]."
→ Tu montres que tu as croisé les sources, ce que les correcteurs valorisent.

5
Je bloque — que faire ? Stratégies de déblocage
Mot inconnu · Notion oubliée · Document inutilisable · Question floue
▶
Cas A — Je ne comprends pas un mot dans la question
Ex : "freins et motivations", "compétences distinctives"…
Stratégie 1
Cherche dans les documents
Les examinateurs donnent souvent des indices dans les documents eux-mêmes. Si le mot est dans un titre de document → lis CE document en premier : il contient la définition ou des exemples.
Stratégie 2
Cherche les notes de bas de page
Les sujets BAC utilisent systématiquement des notes (¹, ², ³) pour définir les mots techniques. Ex. BAC 2025 Bébé Serein — note ¹ : "perturbateurs endocriniens : substances à haut potentiel de risque pour la santé". Vérifie d'abord avant de paniquer.
Stratégie 3
Décompose le mot
"Freins" = ce qui empêche d'acheter. "Motivations" = ce qui pousse à acheter. "Compétences distinctives" = compétences + distinctif = ce que l'entreprise fait différemment des autres. Les mots sont souvent leur propre définition.
Stratégie 4
Réponds sur le thème général
Si tu ne comprends vraiment pas : identifie le thème (RSE ? Ressources ? Stratégie ?) et réponds sur ce thème avec les docs. Un correcteur valorise une réponse partielle bien construite.
Cas B — Je ne me souviens plus de la notion
Ex : j'ai oublié la formule FRNG, je confonds flux tendus et poussés…
Stratégie 1
Cherche dans les questions voisines
Les examinateurs formulent parfois la notion dans la question elle-même. Ex. : "Analyser l'équilibre financier en calculant le fonds de roulement net global (FRNG), le besoin en fonds de roulement (BFR) et la trésorerie nette (TN)" → la parenthèse te rappelle le nom complet et te guide.
Stratégie 2
Cherche dans les documents
Un document de presse ou institutionnel peut contenir la définition de la notion. Ex. : un doc sur la RSE peut rappeler ce qu'est le triple bilan. Le doc financier avec ses colonnes Actif/Passif te permet de retrouver les formules par la structure même du tableau.
Stratégie 3
Raisonne depuis le contexte
Si tu oublies "flux poussés" mais tu sais que Réo doit faire des stocks à cause de l'affinage → c'est la bonne réponse, même sans le terme exact. Le fond compte autant que la forme.
⚠️ Règle d'or : une réponse sans le terme exact mais avec la bonne idée vaut plus qu'une réponse vide.
Comment retrouver les notions souvent oubliées
FRNG / BFR / TN
Le doc financier a des colonnes Actif/Passif → les formules se déduisent de la structure du tableau
Flux tendus vs poussés
Cherche si l'entreprise a des stocks ou produit sur commande → ça te dit le type de flux
Marketing concentré vs différencié
Le doc 1 ou 2 décrit le segment ciblé → s'il y en a qu'un seul = concentré
Compétences distinctives
Cherche dans les docs ce que l'entreprise fait QUE SES CONCURRENTS NE FONT PAS
Démarche qualité (Kaizen, PDCA)
Le doc sur le processus ou la production mentionne souvent les certifications
RSE triple bilan
Le doc 1 ou "engagements" liste les actions sociales + enviro + économiques
Management stratégique vs opérationnel
Stratégique = long terme + ressources importantes + impact sur toute l'organisation
6
La question globale — à quoi elle sert vraiment
La plupart des élèves ne la lisent pas vraiment. C'est une erreur.
▶
La question globale (en gras ou italique au début du sujet) est la problématique centrale. Elle résume l'enjeu de l'entreprise et donne le fil rouge de toutes les questions des 3 dossiers.
Le schéma de toutes les questions globales
"L'entreprise fait face à [un enjeu / un défi concurrentiel]. Les choix qu'elle a faits sont-ils pertinents pour [objectif : se démarquer / assurer sa pérennité / maintenir sa performance] ?"
Exemples tirés des annales réelles
BAC 2025 — Bébé Serein
"Peut-elle se démarquer et accroître sa performance dans un contexte de marché fortement concurrentiel ?"
→ Toutes les questions portent sur comment Bébé Serein se différencie (modèle éco, RSE, diversification lait)
BAC 2024 — Topschuss
"Les choix stratégiques de Topschuss sont-ils pertinents pour maintenir sa croissance ?"
→ Chaque dossier évalue UN choix stratégique (marque employeur, robot Autostore, financement, RSE)
BAC 2023 — Ankama
"Les spécificités d'Ankama lui permettent-elles d'être performante ?"
→ Les questions mesurent la performance de chaque spécificité (DAS, ressources, culture, coût de production)
BAC 2022 — Caraïbes Fumés
"Les choix stratégiques assurent-ils la pérennité de l'entreprise ?"
→ Chaque question évalue si un choix (expansion Martinique, produits, financement) assure la survie
Bac Blanc — Fromagerie Réo
"Les choix de gestion de Réo sont-ils pertinents pour assurer sa performance ?"
→ Chaque question évalue UN choix : qualité AOP, flux poussés, PGI, identité numérique
Les 3 usages concrets de la question globale
Usage 1
Comprendre une question floue
Si une question te semble floue, relis la question globale. Elle te dit dans QUELLE DIRECTION aller.
Q 2.3 Bébé Serein : "Justifier la décision de se positionner sur le lait bio" → relis "comment se démarquer dans un marché concurrentiel ?" → ta réponse doit montrer que le lait bio est une façon de se différencier.
Usage 2
Conclure chaque réponse
Chaque réponse devrait pouvoir se terminer par un lien avec la question globale.
Formule : "Ainsi, [cette décision / compétence / outil] contribue à [reformuler la question globale]."
Ex. : "Ainsi, la démarche qualité AOP permet à Réo d'assurer la pertinence de ses choix et de maintenir sa performance."
Usage 3
Cadrer les questions "15 lignes"
Pour les questions rédigées longues, la question globale est le prisme sous lequel répondre.
"En 15 lignes, montrer que la RSE est créatrice de valeur" → tes exemples doivent montrer que la RSE aide Bébé Serein à se démarquer = répondre à la question globale.
Ce que la question globale révèle : l'entreprise a fait DES CHOIX RÉELS → qui ont des avantages (les questions t'amèneront à les identifier) et des limites (les questions t'amèneront à les nuancer). Toute réponse doit aller dans le sens : "ce choix est pertinent parce que… / mais il a des limites car…"

Les pièges classiques — ce qui fait perdre des points sans raison
Tirés de l'analyse des copies et des grilles de correction BAC
▶
📖
Lire tous les documents avant de regarder les questions
Résultat : tu lis sans savoir quoi retenir, tu oublies tout, et tu dois relire. C'est une perte de temps inutile qui peut désorganiser toute la copie.
✔ Fix : questions EN PREMIER, toujours. Les docs sont lus au fil des questions, pas avant.
💰
Calculer sans interpréter
Donner le résultat d'un FRNG ou d'un taux d'évolution sans dire ce que ça signifie pour l'entreprise. Le correcteur attend TOUJOURS un commentaire — c'est souvent la moitié des points.
✔ Fix : finir chaque calcul par "Ce résultat signifie que…" ou "L'entreprise est donc…"
📄
Répondre en général sans citer le cas
Écrire une définition du cours sans l'appliquer à l'entreprise du sujet. Le correcteur ne peut pas attribuer les points — ton savoir cours ne compte que si tu le connectes au cas.
✔ Fix : chaque argument doit être suivi de "d'après le document X" ou d'un fait tiré du cas (nom de l'entreprise, chiffre, date).
🚫
Ignorer un document listé dans l'énoncé
Si l'énoncé indique "Documents à exploiter : 3, 7, 9" et que tu n'utilises que le 3 et le 9, tu perds des points garantis sur le 7 — même si tu ne vois pas son utilité immédiate.
✔ Fix : si tu bloques sur un document, cite au moins une phrase ("Selon le document 7, …"). Les correcteurs vérifient chaque document listé.
🔀
Confondre les notions proches
Flux tendus ≠ flux poussés · Innovation produit ≠ innovation de procédé · Compétences distinctives ≠ ressources · Marketing concentré ≠ différencié · Ces distinctions tombent souvent au BAC.
✔ Fix : revoir ces paires dans les Flash Cards. Utiliser la Stratégie 3 de la Section 5 pour retrouver depuis le contexte.
🌍
Sur les questions RSE : oublier le lien avec la performance
Décrire les actions RSE de l'entreprise sans expliquer en quoi elles renforcent sa compétitivité ou son image. La RSE seule ne suffit pas — elle doit être reliée à un avantage.
✔ Fix : toujours conclure "…ce qui lui permet de [avantage concurrentiel : image, recrutement, fidélisation, financement]"
💡
Sur le modèle économique : ne pas nommer le type
Expliquer comment l'entreprise gagne de l'argent sans utiliser les termes du cours (freemium, commission, abonnement…). L'examinateur attend d'abord la catégorie, puis l'application.
✔ Fix : toujours commencer par "Il s'agit d'un modèle économique de type [nom]…" puis donner l'exemple de l'entreprise et un exemple hors cas.
📝
Laisser une question sans réponse
Même une réponse partielle ou approximative peut rapporter des points. Une feuille blanche rapporte 0. Le correcteur valorise une réponse bien construite même incomplète.
✔ Fix : s'appuyer sur la Stratégie 4 de la Section 5 — identifier le thème général et répondre en cohérence avec les docs disponibles.
