Bloc 2
Comprendre ce que la question demande vraiment
Toujours lire la question deux fois
Règle clé — lire deux fois
1ère lecture → le VERBE
Qu'est-ce que je dois faire ?
2ème lecture → le COMPLÉMENT
Sur quoi je dois le faire ?
Exemple : "Analysez la validité du contrat conclu entre X et Y à partir des documents."
→ 1ère lecture : "analyser" = vérifier des conditions et conclure
→ 2ème lecture : "validité du contrat" = les 4 conditions de formation du contrat + conclusion

Bloc 3
Lier les questions aux documents
2 règles de base + code des titres de documents
Règles de base
Règle 1 — Lis la question → cherche le document, pas l'inverse
Tu ne lis pas tous les documents d'abord. Tu lis la question, tu identifies ce qu'il te faut, puis tu vas chercher dans les documents.
Règle 2 — Chaque document correspond généralement à 1 ou 2 questions
Le sujet est construit pour que les documents alimentent les questions. Si tu ne trouves pas d'infos utiles dans un document pour une question, c'est probablement qu'il est fait pour une autre question.

Situations particulières
Si aucun document ne correspond à une question
Tu réponds depuis ton cours. Certaines questions ne nécessitent pas de document (ex : "Rappelez les conditions de validité d'un contrat"). Ne cherche pas à tout prix un document.
Si plusieurs documents semblent correspondre à la même question
Tu les utilises tous. Citer plusieurs documents dans une même réponse est valorisé. Formule : "Selon le document 1..." / "Le document 3 précise en outre que..."
Bloc 4
Si je bloque
3 situations, 3 protocoles immédiats
Situation 1
Je ne reconnais pas la notion demandée
1
Relis la question en cherchant le thème (T5/T6/T7/T8) — pas la notion précise.
2
Dans le document, entoure les mots juridiques présents (contrat, responsabilité, salarié, licenciement, société...).
3
Commence à écrire ce que tu sais sur le thème identifié — souvent la notion précise revient pendant la rédaction.
4
Si toujours bloquée : définis les termes de la question mot par mot — souvent, la réponse est dans la définition.
Situation 2
Je n'ai pas assez d'éléments dans les documents pour répondre
1
Vérifie que tu as bien lu TOUS les documents — parfois l'élément est dans un document que tu as survolé.
2
Rappelle-toi : certaines questions se répondent depuis le cours, pas depuis les documents.
3
Si manque réel d'information : dis-le explicitement dans ta réponse ("Les documents ne précisent pas si X a été convoqué par lettre recommandée, or cette condition est requise par le Code du travail..."). Un correcteur valorise le raisonnement même quand les données manquent.
Situation 3
Je n'ai plus le temps pour une question
1
Écris d'abord la qualification et la conclusion (les 2 éléments les plus importants).
2
Ajoute ensuite les éléments de justification dans le temps restant.
3
Ne laisse jamais une question entièrement vide — 2 phrases bien construites valent toujours quelques points.
4
Ne reviens jamais sur une question déjà traitée pour l'améliorer — les points non encore gagnés sur une question vierge sont plus rentables.
Mémo final
Les 6 règles d'or + 5 structures invariables
À connaître par cœur le jour J
Droit STMG — Mémo Jour J
Règle 1
Lis les questions AVANT les documents
Questions d'abord. Toujours. Sans exception.
Règle 2
Verbe = ce que tu dois faire
Analyser → vérifier + conclure · Qualifier → nommer + justifier · Calculer → chiffre + commentaire
Règle 3
Structure = Qualifier → Conditions → Conclure
Toujours dans cet ordre. Sans exception.
Règle 4
Mots du cours = points garantis
"Lien de subordination" · "Cause réelle et sérieuse" · "Fait générateur" · "Lien de causalité" · "Nullité relative" · "Réparation intégrale"
Règle 5
Cite les documents dans ta réponse
"Selon le document 2..." / "Le contrat précise que..." — une réponse sans référence aux docs = réponse de cours
Règle 6
Conclus toujours
"En conclusion, [X] est / n'est pas responsable..." · "En conclusion, la procédure est / n'est pas régulière" — sans conclusion = réponse inachevée
Les 5 structures invariables à connaître par cœur
Validité d'un contrat
4 conditions → application aux faits → conclusion (nul ou valide)
Responsabilité civile
Régime (contractuel / délictuel) → fait générateur + dommage + lien de causalité → exonération ? → conclusion
Licenciement
Fond : motif (cause réelle et sérieuse ?) + Forme : procédure (3 étapes dans l'ordre avec délais) → conclusion
Qualification contrat de travail
Lien de subordination (définition) → indices dans les faits (direction / contrôle / sanction) → conclusion + conséquences si requalification
Choix forme juridique
Critères du sujet (associés, capital, responsabilité souhaitée) → forme adaptée (arguments) → formes exclues (pourquoi) → recommandation
